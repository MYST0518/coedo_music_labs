# COEDO MUSIC LABO

音楽、映像、出版、そして最新の AI 技術をクロスオーバーさせ、次世代のメディア体験をプロデュースするクリエイティブ・ラボ「COEDO MUSIC LABO」の公式サイトです。

## 技術スタック

- HTML5 / CSS3 (Vanilla CSS)
- JavaScript (Vanilla JS)
- Google Fonts (Inter, Noto Sans JP)

## プロジェクト構成

- `index.html`: トップページ
- `music.html`: 音楽制作・出版
- `video.html`: 映像制作・出版
- `books.html`: 書籍制作・出版
- `broadcast.html`: 放送・番組制作
- `events.html`: イベント企画・開催
- `education.html`: 教育事業（療育ワークショップ）
- `produce.html`: プロデュース・仲介業務
- `platform.html`: プラットフォーム開発・運営
- `style.css`: スタイル定義
- `script.js`: インタラクション制御

## 公開方法

このプロジェクトは GitHub Pages で公開されることを想定しています。

1. リポジトリを GitHub にプッシュします。
2. Settings > Pages から、`main` ブランチの `/` ルートを公開対象に設定します。
